th-cal.TK Treibhaus Technik Terminplanung

Über th-cal.TK

th-cal.TK wurde im Oktober 2014 im Auftrag von Tom Neumayr für die Terminplanung der Technik Crew des Treibhaus Innsbruck entwickelt. Das Tool basiert auf OpenSource Frameworks, ist absolut modular und folgt dem MVC-Paradigma.

Version 2.0 (30. Oktober 2014)
Entwicklung Laurin Ostermann
Design Teresa Ostermann, Natja Mohorko
QA Laurin Ostermann, Teresa Ostermann, Daniel Plörer
Basierend auf Bootstrap (v3.1.1), Fat Free Framework (v3.3.0)
